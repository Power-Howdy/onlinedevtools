
---

### 2️⃣ The tools are extremely simple

Most tools are **<100 lines of code**.

Example JSON formatter:

```javascript
function formatJSON(input){
  try {
    return JSON.stringify(JSON.parse(input), null, 2)
  } catch {
    return "Invalid JSON"
  }
}
```

That’s literally the tool.
